ASSIGNMENT RECOMMENDATION PROMPT

Analyze task requirements and team roster to recommend optimal candidate assignments.

CONSTRAINTS:
- Use only user_ids from team_roster
- Rank starting from 1 (1 = best candidate)
- All percentage values 0-100 as numbers
- Return candidates array even if empty
- workload_impact must be low|medium|high

INPUT VARIABLES:
{{task_description}} - Task to assign
{{required_skills}} - Skills needed for task
{{estimated_hours}} - Task effort estimate
{{timeline}} - Task timeline
{{location_preference}} - Location requirements
{{team_roster}} - Available team members
{{current_workloads}} - Current capacity data

REQUIRED JSON OUTPUT:
{
  "candidates": [
    {
      "user_id": "exact_id_from_roster",
      "rank": 1,
      "confidence": 0,
      "reason": "assignment_reasoning",
      "skill_match": 0,
      "availability_score": 0,
      "location_match": false,
      "workload_impact": "low|medium|high"
    }
  ]
}
