GOAL BREAKDOWN PROMPT

Break down strategic goal into concrete, actionable projects with assignments.

CONSTRAINTS:
- Use only user_ids from team_roster
- Keep estimated_hours as numeric values
- Return arrays even if empty
- Dependencies are project titles, not IDs
- Confidence values 0-100

INPUT VARIABLES:
{{goal_text}} - Strategic goal to break down
{{organization_name}} - Organization name
{{team_size}} - Number of team members
{{timeline}} - Project timeline
{{budget_info}} - Budget constraints
{{team_roster}} - Available team members with skills
{{available_skills}} - Skills inventory

REQUIRED JSON OUTPUT:
{
  "projects": [
    {
      "title": "project_title",
      "description": "detailed_project_description",
      "estimated_hours": 0,
      "dependencies": [],
      "acceptance_criteria": [],
      "required_skills": [],
      "suggested_assignees": [
        {
          "user_id": "exact_id_from_roster",
          "reason": "assignment_reasoning",
          "confidence": 0
        }
      ],
      "location_pref": "Remote|Office|Hybrid",
      "priority": "low|medium|high|critical",
      "risk_factors": []
    }
  ]
}
